# Portfolio Artifacts Plan

Use this checklist to prepare proof assets for proposals and case-study pages.

## Required screenshots and demos

1. Workflow architecture map (Lucidchart or equivalent)
   - Caption: `End-to-end onboarding flow with webhook, retry, and fallback lanes.`
2. Zapier scenario screenshot (Typeform -> webhook -> Slack)
   - Caption: `Lead capture automation reducing manual entry during intake.`
3. Make scenario screenshot (qualification and Stripe customer creation)
   - Caption: `High-volume branch with retries and API enrichment for reliability.`
4. Monitoring dashboard screenshot (success/failure trend + uptime)
   - Caption: `Operational dashboard used for 99.9% uptime tracking.`
5. Incident recovery log snippet (anonymized)
   - Caption: `Replay-based recovery using idempotency keys after transient failures.`

## Caption formula

`Context -> What was automated/fixed -> Measurable outcome`

Example: `Lead qualification handoff -> Stripe customer auto-creation with retries -> reduced onboarding cycle from 14 to 3 days.`
